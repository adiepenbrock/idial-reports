Project Pulse built a fully automated processing pipeline for high-frequency sensor data in materials research. By integrating real-time streaming, automatic quality assurance, and FAIR-compliant archiving, manual effort in data curation is reduced by more than 60 percent. The pipeline has been released as open-source software and is now used by four other DFG projects. In the final reporting year, all scientific results were reproducibly documented and the dataset published under a CC-BY license.
